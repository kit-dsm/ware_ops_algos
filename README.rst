ware_ops_algos
==============

What is it?
-----------
A modular Python library of algorithms for problems in warehouse operations, covering item assignment, order batching, picker routing, and scheduling.



``ware_ops_algos`` is designed with a unified interface that enables easy comparison, composition, and extension of algorithms.

Getting Started
---------------

Clone the repository:

.. code-block:: console

    git clone https://github.com/kit-dsm/ware_ops_algos.git
    cd ware_ops_algos

Install the project in editable mode:

.. code-block:: console

    pip install -e .
